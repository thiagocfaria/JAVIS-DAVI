# Bench Audio (pasta enxuta)

- Arquivos canônicos de áudio para benchmarks:
  - `voice_clean.wav` — fala limpa (16 kHz mono).
  - `voice_noise.wav` — fala com ruído (16 kHz mono).
- Artefatos históricos (JSON) vão para `bench_audio/archive/` (não são usados pelos docs atuais).
- Para novos benchmarks, salve apenas o mínimo necessário (ex.: 1 JSON por cenário com data) ou arquive em `archive/` para não poluir a pasta raiz.
